# Laboratorium 1: Git, GitHub i przygotowanie środowiska Django
SYMULACJA KONFLIKTU

Celem tego laboratorium jest **praktyczne przygotowanie środowiska programistycznego, opanowanie systemu kontroli wersji Git oraz integracja z platformą GitHub wraz z automatyzacją.**

W ramach realizacji zadań i wymagań zaliczeniowych wykonano następujące kroki:
* Skonfigurowano bezpieczne połączenie z GitHubem przy użyciu kluczy SSH.
* Zainicjalizowano projekt i poprawnie skonfigurowano plik `.gitignore`..
* Przećwiczono pracę z repozytorium na wielu gałęziach (dwie dodatkowe gałęzie oprócz głównej).
* Wykorzystano formatowanie Markdown do stworzenia czytelnej dokumentacji - `python manage.py runserver` .
* Skonfigurowano narzędzie GitHub Actions do automatycznej weryfikacji kodu (testy workflow).

Link do [oficjalnej dokumentacji Django](https://docs.djangoproject.com/).
